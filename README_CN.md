## Merlin 游戏
- 成为你自己的魔法师 🧙
- 探索**多样**的魔法元素，🪄 创造**属于你**的法术
- 穿越**多样的魔法世界**，**击败**危险的敌人 🦹

### 如何游玩
- 从**release**下载游戏文件 `Merlin_v0.1.0_Windows.zip`
- 解压游戏文件并运行 `Merlin.exe` 开始你的冒险！（需要Windows操作系统）
- 我们在登录界面提供了详细的游戏说明👇
